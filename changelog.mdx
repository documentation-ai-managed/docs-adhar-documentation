<Update label="2024-11-15" description="v2.0.0" tags={["feature", "breaking"]}>

## Breaking Changes

- Updated MDX parsing engine - Review your custom components for compatibility.

## New Features

- Added `<Video>` and `<Iframe>` components for rich media embeds.
- Introduced `<Columns>` with up to 5 flexible grid options.
- Enhanced `<ParamField>` with `deprecated` attribute support.

## Improvements

- Faster build times by 40% with optimized JSX escaping.
- Better error messages for common parsing issues like unescaped `{` characters.

## Bug Fixes

- Fixed nested code block rendering with 4-backtick support.
- Resolved heading hierarchy warnings in validation.

</Update>

## Upgrade Guide

Follow these steps to update your Adhar Documentation site. Test in a staging environment first.

<Steps>
  <Step title="Backup" icon="database">
    Save your current `docs/` directory.
  </Step>
  <Step title="Update Package" icon="package">
    Run the upgrade command for your platform.
    
    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install adhar-docs@latest
        ```
      </Tab>
      <Tab title="yarn" icon="package">
        ```yarn
        yarn add adhar-docs@latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Rebuild" icon="hammer">
    ```bash
    npm run build
    ```
    
    Verify no parsing errors in the console.
  </Step>
  <Step title="Deploy" icon="upload-cloud">
    Push changes to your hosting provider.
  </Step>
</Steps>

<CodeGroup tabs="JavaScript,TypeScript">

  // Example: New Video component usage
